"St-st-st-Stutter Midi Effect"

This is a Web based MIDI plug-in that takes the pitch and velocity values from a MIDI input signal and creates a stutter effect based on the notes played and the adjustable parameters seen below. You can adjust the rate at which the signal stutters, how long the stutters last, and how much the velocity decreases with each stutter.
